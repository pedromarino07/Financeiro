# Gerenciador de Finanças Pessoais

Este é um sistema web completo para o controle financeiro pessoal, permitindo o registro de receitas, despesas e a visualização do saldo total. O projeto foi desenhado para ser performático, seguro e utilizar tecnologias modernas de desenvolvimento web.

## 🚀 Sobre o Projeto

Este sistema foi idealizado e desenvolvido como parte do portfólio de engenharia de software de Pedro Marino Viana Lima. O foco principal é demonstrar o domínio da integração entre o frontend web e um banco de dados relacional robusto.

A aplicação permite:
* Cadastro e categorização de movimentações financeiras.
* Visualização de fluxo de caixa (entradas e saídas).
* Consulta de saldo atualizado em tempo real.

---

## 🛠️ Tecnologias Utilizadas

O projeto utiliza a stack JavaScript completa, garantindo homogeneidade no código.

* **Banco de Dados:** [PostgreSQL](https://www.postgresql.org/) (Modelagem relacional para consistência de dados).
* **Backend:** [Node.js](https://nodejs.org/) com o framework **Express**.
* **Frontend:** HTML5, CSS3 e JavaScript Vanilla (para demonstração de manipulação pura do DOM).
